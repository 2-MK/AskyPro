# Matrix Vision

Matrix Vision is a web application that converts live webcam footage into real-time ASCII art with a Matrix-inspired interface.

## Features

- Live webcam to ASCII conversion
- Selfie mode (mirrored camera)
- Matrix-themed UI
- Green terminal-style design
- Responsive fullscreen layout

## Technologies Used

- HTML
- CSS
- JavaScript
- Canvas API
- Web Camera API (getUserMedia)

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/matrix-vision.git
```

2. Open the project in VS Code.

3. Install the Live Server extension.

4. Right-click `index.html` and select **Open with Live Server**.

5. Click **INITIALIZE CAMERA** and allow camera access.
